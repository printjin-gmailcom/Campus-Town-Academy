
고객 세분화

- 고객 등급, 인구 통계적 구분, RFM

> 알고리즘보단 데이터 탐색 및 직관에 기반 (논리적이여야함)



EDA를 통한 고객 세분화

- 목적에 맞게 분석가 직관에 따름 / 세그먼트 이해 쉬움 / 세그먼트 수정 편리 / 전략 짜기 쉬움

- 상호배타적이며 전체를 포괄한, 의미가 명확한 환경 변화에도 유지되는 세분화가 좋은 세분화

- 세분화 목적 정의 -> 목적에 맞는 개념 확인 -> 개념을 반영할 수 있는 데이터 확인 -> 변수의 분포 확인 후 분할 지점 정의 -> 각 세그먼트 별 비율 확인 -> 세그먼트 별 프로파일링

- 가치 기반 세분화 / 인구 통계학적 세분화 / 행동 기반 세분화 / 가치 기반 확장 세분화(RFM) 등



분석 프로세스

- 데이터 품질 확인 -> EDA를 통한 주요 정보 확인 -> RFM 스코어 생성 -> RFM 모형 단순화 -> 프로파일링





E-commerce Sales Forecast 데이터 설명

1 InvoiceNo 주문번호 숫자형 

2 StockCode 상품 코드 문자형 

3 Description 상품 설명 명목형 

4 Quantity 주문수량 연속형 

5 InvoiceDate 주문일시 명목형 

6 UnitPrice 상품가격 연속형 

7 CustomerID 고객 ID 연속형 

8 Country 국가명 명목형
